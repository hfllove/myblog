# scroll-view 组件

基于以上实例，实现纵向，横向滚动的效果

1. 实现纵向滚动

scroll-view 组件通常搭配 scroll-y 组件属性，来实现纵向滚动的效果，以下是纵向滚动的 scroll-view 实例部分内容：

  • 模板代码 WXML

scroll-y: 允许纵向滚动
scroll-x: 允许横向滚动

注意

使用纵向滚动时，必须给 scroll-view 一个固定高度

<scroll-view class="container1" scroll-y>
  <view>A</view>
  <view>B</view>
  <view>C</view>
</scroll-view>

# swiper 和 swiper-item 组件

参考官方文档 组件--视图容器--swiper组件 (opens new window)

# swiper常用属性

  • indicator-dots 是否显示面板指示点
  • indicator-color 指示点颜色
  • indicator-active-color 当前选中的指示点颜色
  • autoplay 是否自动切换
  • interval 自动切换时间间隔
  • circular 是否采用衔接滑动

# 实例

  • WXML
<!-- 轮播图的结构 -->
<swiper class="swiper-container" indicator-dots="true" indicator-color="white" 
indicator-active-color="gray" autoplay interval="3000" circular="true">
  <!-- 第一个轮播图 -->
  <swiper-item>
    <view class="item">A</view>
  </swiper-item>
  ...
</swiper>
